# 育児シェア アプリ プライバシーポリシー

最終更新日：2026年1月

## 収集する情報
本アプリは、ユーザーが入力した育児記録データを保存するために Firebase を利用します。

以下の情報を保存する場合があります：
- アカウント情報（Apple ID / メールアドレス）
- 表示名
- 家族グループID
- 育児記録データ（ユーザーが入力した内容）

## 利用目的
収集した情報は、以下の目的でのみ利用されます：
- アプリの機能提供
- データの保存および同期
- ユーザー認証

## 第三者提供
取得した情報を第三者に提供することはありません。

## データの管理
Firebase（Google LLC）が提供するクラウドサービスを利用して、安全にデータを管理します。

## アカウント削除
ユーザーはアプリ内の設定画面から、アカウントおよび関連データを削除できます。
